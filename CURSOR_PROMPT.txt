## TECH REQUIREMENTS

**Core:**
- Next.js 15 with App Router (TypeScript strict mode)
- React 18+ with Server Components

**Styling:**
- Tailwind CSS 4
- shadcn/ui components
- Framer Motion for animations
- Lucide React icons
- Design system from spec (Lines 17-135)

**State & Data:**
- Zustand (client state)

- React Hook Form

- NextAuth.js v5
- Phone verification (SMS mock initially)
- JWT with refresh tokens

**Other:**
- Stripe for payments (test mode)
- uploadthing for file uploads
- react-pdf for document viewing
- signature_pad for e-signatures
