### Component Styles

**Buttons:**
```css
/* Primary Button */
.btn-primary {
  background: var(--primary-blue);
  color: white;
  padding: 12px 24px;
  border-radius: 8px;
  font-weight: 600;
  transition: all 0.2s;
}
.btn-primary:hover {
  background: var(--primary-blue-dark);
  transform: translateY(-1px);
  box-shadow: 0 4px 12px rgba(0, 102, 255, 0.3);
}

/* Secondary Button */
.btn-secondary {
  background: white;
  color: var(--primary-blue);
  border: 2px solid var(--primary-blue);
  padding: 12px 24px;
  border-radius: 8px;
  font-weight: 600;
}
```

**Input Fields:**
```css
.input {
  padding: 12px 16px;
  border: 1px solid var(--neutral-300);
  border-radius: 8px;
  font-size: var(--text-base);

}
.input:focus {
  border-color: var(--primary-blue);
  outline: none;
  box-shadow: 0 0 0 3px var(--primary-blue-light);
}
.input.error {
  border-color: var(--error-red);
}
```

**Header:**
- Logo: Left-aligned, links to home
- Navigation: PERSONAL, BUSINESS, PARTNERSHIPS, COMPANY
- CTAs: "MY ACCOUNT" (outline button), "GET COVERED" (solid button)
- Sticky on scroll

**Hero Section:**
- Headline: "GENERAL LIABILITY? WE'VE GOT YOU COVERED" (Large, bold)
- Animated pink circle behind "E" in "GENERAL"
- Subheading: "Personalized coverage for you and your business starts here"
- Instant quote indicators on the right (animated)

**Value Proposition Cards:**
- Two equal-width cards side-by-side
- Background images with gradient overlay
- Card titles and descriptions
- Two CTA buttons each: "SPEAK TO AGENT" (outline), "GET COVERAGE" (solid)
- Hover effect: subtle lift and shadow

**Social Proof Section:**
- Heading: "Trusted by thousands, loved by individuals"
- 4 testimonial cards in a row
- Each card: Avatar, Name, Title, 5-star rating, Review text
- Carousel functionality on mobile

## Mobile Considerations

### Responsive Breakpoints

```css
/* Mobile First Approach */
- Mobile: 320px - 767px
- Tablet: 768px - 1023px
- Desktop: 1024px+
```

### Mobile-Specific Adjustments

**Navigation:**
- Hamburger menu instead of full nav
- Bottom navigation bar for key actions
- Swipe gestures for step navigation

**Forms:**
- Single column layout
- Larger touch targets (min 44px)
- Native input types (tel, email, number)
- Floating labels to save space

**Quote Comparison:**
- Horizontal swipe carousel
- Sticky "Compare" button at bottom
- Comparison view as full-screen modal

**Chat:**
- Full-screen on mobile
- Keyboard-aware scrolling
- Quick reply suggestions

**Dashboard:**
- Stacked cards instead of grid
- Collapsible sections
- Pull-to-refresh

## Accessibility

### WCAG 2.1 AA Compliance

**Color Contrast:**
- Text: Minimum 4.5:1 ratio
- Large text: Minimum 3:1 ratio
- Interactive elements: Minimum 3:1 ratio

**Keyboard Navigation:**
- All interactive elements focusable
- Visible focus indicators
- Logical tab order
- Escape key closes modals

**Screen Readers:**
- Semantic HTML (headings, landmarks)
- ARIA labels for icons and buttons
- ARIA live regions for dynamic content
- Alt text for all images

**Forms:**
- Proper label associations
- Error messages linked to fields
- Required field indicators
- Success confirmations
